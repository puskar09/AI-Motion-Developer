## Stack

Python. OpenAI and Anthropic APIs. LangChain when it makes
sense, raw API calls when it doesn't. LangGraph for agents
that need a brain. CrewAI when one agent isn't enough.
ChromaDB and Pinecone for memory. Ollama when I want it
running on my machine with no internet and no bill.


## Where this is going

The goal is to go from calling an API to building systems
where multiple AIs work together like a team — each one
with a role, tools, and memory.

By the end of this repo I want to have built something
I could actually charge money for.

That's the only benchmark that matters.
